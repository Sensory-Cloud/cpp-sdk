Detailed Description

template<typename Factory, typename Request, typename Response>
struct sensory::calldata::CallbackData< Factory, Request, Response >

A type for encapsulating data for asynchronous calls.

Template Parameters
FactoryThe factory class that will manage the scope of the call.
RequestThe type of the request message.
ResponseThe type of the response message.

The Factory is marked as a friend in order to provide mutable access to private attributes of the structure. This allows instances of Factory to mutate to the structure while all external scopes are limited to the immutable interface exposed by the public accessor functions. Instances of CallbackData are mutable within the scope of Factory, but immutable outside of the scope of Factory.
